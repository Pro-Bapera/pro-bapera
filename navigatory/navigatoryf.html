◣Фрейм_коллекции_справка

Сюда, по клику на ※-кнопку (во фрейме слева) перед названием какой-то коллекции, загружается список экспонатов этой коллекции, со ссылками на страничные интернет-ресурсы (литературные-научные-технические-музыкальные опусы html-pdf-djvu-mp3 формата – экспонаты коллекции).

◢Локализация указанных интернет-ресурсов на своём компьютере

◢ Концепция локализации

Образ данного i-Net навигатора позволяет использовать ссылку на интернет-ресурс для обращения к локализации данного интернет-ресурса на локальном Windows IIS веб-сервере по тому же интернет-адресу интернет-ресурса с заменой http://... на http://localhost/... (соответствующий исходно папке C:\inetpub\wwwroot\).

Каждый может по-своему организовывать структуру локализации сохранённого из интернета; в данном случае предлагается использовать готовый i-Net навигатор с поддержкой локализации и существующую web-структуру интернета.

Такой i-Net навигатор унифицированное средство для навигационного покрытия всего ресурсно-полезного в интернете на основе общепринято-единых и личных предпочтений.

Остаётся настроить IIS веб-сервер и автоматизировать сохранение, части или всего указываемого на странице навигатора, на соответствующем месте, посредством бесплатного DownloadMaster.

◢Оперирование во фрейме коллекции

Если ссылочные интернет-данные ещё не сохранялись самим пользователем, то, при наведении указателя мышки на гиперссылку, в статус-строке визуализируется подсказка: куда и под каким именем сохранять данные.

Если ссылочные интернет-данные сохранены на указанном месте, то, при наведении указателя мышки на гиперссылку, тут же визуализируется маленькая кнопочка обращения к локализованной копии, открываемой в плавающем окне (плавающие окна должны быть разрешены для localhost)

Читаемые-исследуемые книги-документы, зачастую объёмны, требуют многократного обращения к ним, поэтому временное сохранение таковых на компьютере пользователя вполне оправдано, однако, корректно-правовое обращение с ними требует их удаления с компьтера по завершению прочтения-исследования. Именно для этого, при наведении указателя мышки на гиперссылку сохранённых данных, в статус-строке визуализируется подсказка: где и под каким именем данные сохранены, чтобы видеть что и где удалять.

◢Дополнительное плавающее окно коллекции

Можно открыть слева на треть ширины экрана дополнительное плавающее окно коллекции (удваивает объём загрузки, что существенно для n-мегобайтного списка коллекции) и смотреть экспонат (единицу коллекции) в открытом справа на две трети ширины экрана плавающем окне.

Можно разместить плавающее окно экспоната слева на две трети ширины экрана и использовать (в трёхчастном окне) для выбора экспоната самый правый фрейм списка экспонатов коллекции.

Плавающее окно mp3-проигрывателя можно переразмерить в линейку внизу экрана.

◢Дополнительно к настройке IIS веб-сервера

Типовое win-размещение под localhost: C:\inetpub\wwwroot\, однако конкретизировав пользователя-администратора можно использовать любое размещение вне раздела с операционной системой (например, D:\www\).

Чтобы на локальном IIS-7 веб-сервере отыскивались сохранённые djvu-формы, необходимо в "Администрирование.Диспетчер служб IIS" прописать IIS MIME "*" с типом "application/octet-stream"
